### Setup

To build all apps and packages, run the following command:

```
pnpm install
pnpm build
```

### Develop

To develop all apps and packages, run the following command:

```
pnpm dev
```

#### Adding new package
To add new package run following command:

```
pnpm run gen:workspace
```

### Test

To run tests

```
pnpm test
```
